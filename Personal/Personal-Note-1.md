This is the first release of the **Task Scheduler C++** project. It establishes the foundation of the repository, including the project structure, documentation, and development workflow.

This version focuses on preparing a professional open-source environment for future development. No scheduler functionality is implemented yet.

**Highlights**

- Clean modular C++ project structure (`src/`, `include/`, `tests/`, `examples/`, `docs/`)
    
- Added essential open-source files (README, LICENSE, CONTRIBUTING, CHANGELOG)
    
- Introduced Semantic Versioning and Conventional Commits workflow
    
- Basic Makefile build setup
    
- Defined roadmap toward **v1.0.0**
    

Future releases will begin implementing the core components, starting with the **thread pool in v0.2.0**.
